# 実装計画: 移動平均線の途切れ修正

## 概要
移動平均線（MA）のデータに欠損がある場合や、計算期間不足で初期データがnullの場合に、線が断続的になる問題を修正する。

## 変更内容

### app.js
- Plotlyのトレース設定（MA5, MA25, MA75）に `connectgaps: true` を追加する。
  - これにより、null値をスキップして線をつなぐようになる。

## 検証計画
### 自動テスト
- なし（UIの描画確認が必要なため）

### 手動検証
- ブラウザでチャートを表示し、移動平均線が途切れずに描画されていることを確認する。
